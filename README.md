# House Price Prediction

This is a beginner machine learning project using Python.

The goal of this project is to predict house prices using a housing dataset.

## Project Steps

- Load the dataset
- Remove missing values
- Split the data into training and testing sets
- Apply log transformation
- Convert categorical data using one-hot encoding
- Create new features
- Train Linear Regression model
- Train Random Forest model
- Use GridSearchCV to improve the model
- Evaluate model performance

## Libraries Used

- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python house_price_prediction.py
```
